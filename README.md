# Kali Linux
```
• sudo apt-get install python2 && git -y
• sudo git clone https://github.com/Alienrazor/Crack
• sudo apt-get install python3-pip
• sudo pip3 install requests mechanize tqdm
• cd Crack
• chmod +x *
• sudo ./main.py
```
